# 自动更新角色卡

- **作者:** 青空莉想做舞台少女的狗
- **版本:** 2025/02/11
- **源文件:** `https://github.com/StageDog/tavern_resource/tree/main/src`

## 功能说明

这个脚本可以让玩家在酒馆中直接更新角色卡，不需要去帖子查看更新，不需要再下载、导入新角色卡。

## 使用方法

### 1. 配置变量

在酒馆助手 "角色脚本" 中添加此脚本，编辑脚本，在 `变量列表` 中填入：

```yaml
{
  角色卡名称: 玄天大陆
  角色卡链接: https://testingcf.jsdelivr.net/gh/你的用户名/你的仓库/玄天大陆.png
  更新日志链接: https://testingcf.jsdelivr.net/gh/你的用户名/你的仓库/更新日志.md
}
```

**注意：**

- 角色卡名称要和角色卡在酒馆中的名称一样，确定好后不能再更改
- 角色卡链接和更新日志链接必须是固定的，而且访问的是最新版本

### 2. 创建更新日志

更新日志格式如下：

```md
# 更新日志

## 这里填版本号，最上面是最新版本

- 最新版本更新了啥……

## 1.0.0

- 这个版本更新了啥……
```

### 3. 上传文件

将角色卡和更新日志上传到 GitHub，使用 jsdelivr CDN 获取固定链接：

- 格式：`https://testingcf.jsdelivr.net/gh/用户名/仓库名/文件路径`

## 工作原理

1. 脚本启动时，从更新日志链接获取最新版本号
2. 与当前角色卡版本进行比较
3. 如果有新版本，显示「更新角色卡」和「更新日志」按钮
4. 点击按钮即可更新或查看更新内容

## 配合 VSCode 实时编写

可以配合 `实时编写角色卡` 功能，在 VSCode 中修改角色卡，上传到 GitHub，玩家在酒馆中自动收到更新提示。
